# Google Analytics 4 - Настройка

## ✅ Статус: ПОДКЛЮЧЕН И АКТИВЕН

Google Analytics 4 успешно настроен и работает на сайте.

## Measurement ID

**ID:** `G-TQVEB29TNF`

## Что отслеживается

### Автоматические события:
- **page_view** - просмотры страниц
- **session_start** - начало сессий
- **first_visit** - первые визиты
- **scroll** - прокрутка страниц

### Кастомные события:
- **Button / Click / Hero CTA - Order** - клик на кнопку "Заказать" в Hero
- **Button / Click / Hero CTA - Portfolio** - клик на кнопку "Портфолио" в Hero
- **Button / Click / Service Order - [название услуги]** - клики по кнопкам заказа услуг
- **Form / Submit / Contact Form Success** - успешная отправка формы

## Где смотреть статистику

1. Зайдите на https://analytics.google.com
2. Выберите свойство NEUROCRAFTS
3. Перейдите в раздел "Отчеты"

### Основные отчеты:
- **Реальное время** - активные пользователи сейчас
- **Сводка** - общая статистика
- **События** - все отслеживаемые события
- **Источники трафика** - откуда приходят пользователи

## Важно

- События начнут появляться в течение 24-48 часов после первых визитов
- Данные в реальном времени доступны сразу
- Для проверки работы откройте сайт в режиме инкогнито и выполните действия

## Дополнительная настройка

Если нужно добавить больше событий, используйте функцию:

```javascript
window.trackEvent('Category', 'Action', 'Label');
```

Примеры:
```javascript
window.trackEvent('Video', 'Play', 'Hero Video');
window.trackEvent('Link', 'Click', 'Footer - Telegram');
window.trackEvent('Download', 'Click', 'Price List PDF');
```
