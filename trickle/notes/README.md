# NEUROCRAFTS Landing Page

Лендинг фрилансера по AI-дизайну в стиле **BRUTAL MINIMALISM** (черно-белый + желтый акцент).

## Стиль дизайна: Brutal Minimalism

- **Цветовая палитра**: Черный + Белый + Желтый акцент (#FFD600)
- **Типографика**: Жирные заголовки (font-black, uppercase), резкие линии
- **Границы**: Толстые 4px рамки, прямые углы
- **Анимации**: Минимальные, быстрые
- **Подход**: Прямота, четкость, без лишних эффектов

## Основные возможности

- **Brutal Hero**: Черно-белый герой с крупной типографикой
- **Box-style Services**: Три услуги в жестких рамках
- **List Gallery**: Проекты списком с крупными номерами
- **Benefits**: Секция преимуществ фрилансера
- **Sharp Forms**: Контактная форма с резкими углами
- **Mobile menu**: Адаптивное бургер-меню
- **Lazy loading**: Отложенная загрузка изображений
- **SEO**: sitemap.xml, Schema.org разметка, meta-теги
- **Preloader**: Быстрая загрузка
- **Scroll indicator**: Индикатор прогресса
- **Form submission**: EmailJS интеграция
- **Performance**: Легкий вес, быстрая загрузка
- **Sidebar**: Боковая навигация
- **Analytics**: Google Analytics готов

## Структура проекта

- `index.html` - главная страница
- `app.js` - основное приложение
- `components/` - UI компоненты (HeroBrutal, Services, Gallery, Benefits, ContactForm, Footer, Sidebar)
- `utils/` - вспомогательные функции (preloader)
- `admin.html` + `admin-app.js` - админ-панель
- `before-after.html` - страница редизайна

## Технологии

- React 18 (production)
- TailwindCSS (brutal utility classes)
- EmailJS (форма)
- Lucide Icons (статичные)

## Дизайн-система

**Цвета:**
- Основной: #000000 (черный)
- Вторичный: #FFFFFF (белый)
- Акцент: #FFD600 (желтый)

**Типографика:**
- Заголовки: font-black (900), uppercase, tracking-tight
- Текст: font-bold (700), leading-tight
- Размеры: крупные (text-5xl - text-8xl для H1/H2)

**Границы:**
- Толщина: 4px (border-4)
- Стиль: прямые углы, без скруглений
- Использование: везде (кнопки, блоки, изображения)

**Принципы:**
- Максимальная простота
- Черно-белая база + желтый акцент
- Крупная типографика
- Толстые рамки
- Минимум анимаций
- Прямота и честность
